Luau API Documentation

Наш API позволяет создавать кастомные модули с нативным интерфейсом Vantality.

Vantality.CreateSection(name)

Создает новую визуальную секцию в окне чита.

example.lua
-- Пример создания простого тоггла
local UI = Vantality.CreateSection("My Script")

UI:AddToggle({
    Name = "Auto Farm",
    Default = false,
    Callback = function(state)
        print("Farm state: ", state)
    end
})

Rules & Requirements

  • 1. Запрещено публиковать вредоносный код (Logger/Stealer).
  • 2. Скрипт должен иметь описание и актуальную версию.
  • 3. Обязательно используйте OnUnload для очистки соединений.
  • 4. Плагиат чужих работ без указания авторства ведет к бану в маркете.
