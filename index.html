1. Начало работы

Все скрипты выполняются в изолированной песочнице (Sandbox). Вам доступен глобальный объект Vantality и стандартные библиотеки Roblox (task, spawn, math и др.).

Технический нюанс: API автоматически сохраняет значения элементов по их Flag или Name в папку configs.

2. Методы Интерфейса (UI)

Vantality.CreateSection (title: string) returns: SectionObject

Создает новую визуальную группу элементов в правой части интерфейса.

Объекты элементов (Add...)

AddToggle(config)

UI:AddToggle({ Name = "Double Jump", Default = false, Risky = false, Option = true, Callback = function(v) end })

Флаг Option = true добавляет кнопку настроек (шестеренку). Доступ к вложенному меню через .Option.

AddSlider(config)

UI:AddSlider({ Name = "Speed", Range = {16, 100}, Default = 16, Precise = 1, Step = 0.5, Type = "studs", Callback = function(v) end })

Precise определяет количество знаков после запятой. Type — суффикс значения.

AddDropdown(config)

UI:AddDropdown({ Name = "Mode", Values = {"Legit", "Rage"}, Multi = false, Callback = function(v) end })

Если Multi = true, Callback вернет таблицу { ["Value"] = true/false }.

AddColorPicker(config)

UI:AddColorPicker({ Name = "ESP Color", Default = Color3.new(1,0,0), Transparency = 0, Callback = function(clr, alpha) end })

3. Системные функции

Vantality.Bind (signal: RBXScriptSignal, callback: function)

Критически важный метод. Подключает функцию к сигналу (например, Heartbeat). Автоматически отключает все коннекты при выгрузке скрипта, предотвращая лаги.

Vantality.Notify (data: table)
Vantality.Notify({ Title = "Alert", Content = "Script Loaded!", Icon = "check", Duration = 5 })

Использует встроенную систему уведомлений с иконками Lucide.

4. Выгрузка (OnUnload)

Если вы определите в своем скрипте глобальную функцию OnUnload, она будет вызвана при нажатии пользователем кнопки "Unload" или при удалении файла.

function OnUnload()
    print("Очистка ресурсов...")
    workspace.CurrentCamera.FieldOfView = 70
end
Правила публикации
  • Никакой обфускации. Весь код в маркетплейсе должен быть читаемым для модерации.
  • Запрещены любые HTTP-запросы на сторонние логгеры (Webhook и т.д.).
  • Скрипт не должен портить FPS пользователям (используйте Vantality.Bind).
