Documentation

Инициализация

Vantality.CreateSection (title: string) returns: SectionProxy

Создает секцию в одной из свободных колонок (Center/Right). API автоматически балансирует количество элементов между колонками для эстетичного вида.

local UI = Vantality.CreateSection("Combat Module")

Элементы интерфейса

Каждый элемент (кроме кнопок) автоматически регистрируется в системе конфигов. Ключом для сохранения является Flag, а если он не указан — Name.

AddToggle

Создает выключатель.
Аргумент Risky окрашивает текст в оранжевый.

UI:AddToggle({ 
    Name = "Auto Block", 
    Default = false, 
    Option = true, -- Добавляет кнопку шестеренки
    Callback = function(v) print(v) end 
})

AddSlider

Ползунок для чисел.
Precise — кол-во знаков после запятой.

UI:AddSlider({ 
    Name = "Power", 
    Range = {1, 100}, 
    Precise = 0, 
    Step = 1,
    Callback = function(v) end 
})

AddDropdown

Список. В режиме Multi возвращает таблицу ключей со значениями true/false.

UI:AddDropdown({ 
    Name = "Select Hero", 
    Values = {"Saitama", "Garou", "Genos"}, 
    Multi = false,
    Callback = function(v) end 
})
Вложенные меню (Proxy Objects)

Методы AddToggle, AddSlider, AddDropdown и AddColorPicker возвращают объект, содержащий поле .Option. Это позволяет добавлять элементы внутрь выпадающего меню этого элемента (кнопка шестеренки).

local main = UI:AddToggle({Name = "Master", Option = true})
main.Option:AddSlider({Name = "Sub Settings", Range = {0, 10}})

Системные методы

Vantality.Bind (signal, callback)

Безопасное подключение к сигналам (RenderStepped, Heartbeat). Обязательно к использованию: все бинды, созданные через этот метод, автоматически отключаются при выгрузке скрипта.

Vantality.Notify (config)

Выводит красивое уведомление в углу экрана. Поддерживает иконки из Lucide.

Vantality.Notify({ Title = "Core", Content = "Loaded!", Icon = "shield", Duration = 3 })

Выгрузка скрипта (Cleanup)

Для предотвращения утечек памяти и артефактов в игре, всегда определяйте функцию OnUnload. Она вызывается ядром API в момент остановки вашего скрипта.

example_cleanup.lua
function OnUnload()
    -- Возвращаем настройки игры в дефолт
    game.Lighting.Brightness = 2
    Vantality.Notify({ Title = "Vantality", Content = "Script Cleanup Done" })
end

local UI = Vantality.CreateSection("Advanced API")

    Name = "Kill All Players",
    Risky = true,

        _G.Loop = state
    end

KillAll.Option:AddSlider({
    Name = "Kill Distance",
    Range = {10, 500},
    Default = 100,
    Type = " studs"
})

Vantality.Bind(game:GetService("RunService").Heartbeat, function()
    if _G.Loop then
        -- логика тут
    end
end)

function OnUnload()
    _G.Loop = false
end
